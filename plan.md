# 3-Month Plan: Clear NQT & Get Job Ready

## PRIORITIES

1. **NQT Prep (50%)** - 50 MCQs + 2 Coding + Cognitive + English
2. **DSA Practice (30%)** - Technical rounds
3. **Internship (15%)** - Complete tasks, get certificate
4. **English (5%)** - Basic proficiency

---

## 3-MONTH TARGETS

### Month 1: Foundation
- Complete WebForms assignment
- Solve 80 DSA problems (easy level)
- Cover OOPS, OS, DBMS basics
- Complete 6 full NQT mock tests
- Practice cognitive (aptitude + reasoning)

### Month 2: Intensive Practice
- Solve 140 more DSA problems (medium level)
- Complete 20 total NQT mock tests
- Revise all technical MCQ topics
- Achieve 70%+ in mock tests
- Strengthen weak cognitive areas

### Month 3: Polish & Peak
- Reach 250+ total DSA problems
- Complete 35+ total mock tests
- Achieve consistent 80%+ scores
- Fix remaining weak areas
- Practice timed coding questions

---
